Deploy Static Website on AWS

In this project,  a static website was deployed to AWS using S3, CloudFront, and IAM.

The CloudFront endpoint for the project is: 

https://d2c88m04hjqqqz.cloudfront.net/index.html

The bucket website endpoint for the project is :
 
http://my-travel-blog-bucket.s3-website-us-east-1.amazonaws.com


Steps followed for the project.

An S3 bucket was created with the name "my-travel-blog-bucket".
It was then made public and then secured via IAM.
It was configured and a bucket website endpoint was generated.
A CloudFront endpoint was then generated to secure the website.
